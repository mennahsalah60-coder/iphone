# PhoneShop

## Project Overview  
This is a frontend web page (static site) named **PhoneShop** — part of a practice project — built using HTML, CSS and (optionally) JavaScript. The site displays a product page (or product list) for iPhone or phones.  

## Purpose  
- To practice building a clean and responsive product page using only front-end technologies.  
- To demonstrate layout skills, styling and responsiveness (mobile / desktop) using basic web technologies.  

## Technologies Used  
- HTML  
- CSS  
- JavaScript  

## What’s Included  
- A product listing / product details page (or homepage) for “PhoneShop / iPhone”  
- Responsive design to look good on different devices  
- Clean and organized code structure
